
# Usage
Clean output directory
```bash
make clean
```

Build a `.txt` version of the CV
```bash
make build-txt
```

Build a `.html` version of the CV
```bash
make build-html
```

Build a `.pdf` version of the CV
```bash
make build-pdf
```

Build a `.docx` version of the CV
```bash
make build-docx
```

Build all the formats. (Cleans the output directory first)
```bash
make build
```
